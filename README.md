Gerenciador de Tarefas (ToDo List)Este projeto é um sistema de gerenciamento de tarefas desenvolvido como Trabalho Acadêmico para a disciplina de Fundamentos de Programação Web (2026). 
Ele permite cadastrar, visualizar, pesquisar, editar e excluir tarefas do dia a dia diretamente no navegador.🚀 Funcionalidades (Requisitos do Sistema)O projeto atende integralmente aos requisitos solicitados:

I.Cadastro de Tarefas: Registro obrigatório de título, descrição, data de criação, prioridade (Baixa, Média ou Alta) e status (Pendente ou Concluída) .
II. Consulta e Pesquisa: Listagem completa das tarefas com funcionalidade de busca por título em tempo real.
III. Gestão de Dados (CRUD):Marcar tarefas como concluídas.Editar o conteúdo de tarefas existentes.Excluir tarefas com confirmação.
IV. Persistência: Armazenamento seguro utilizando LocalStorage, garantindo que os dados não sejam perdidos ao atualizar a página.
V. Validação: Implementação de travas para impedir o cadastro de campos vazios.VI. Feedback ao Usuário: Exibição de mensagem informativa caso nenhuma tarefa seja encontrada na lista ou na busca.

Interface e DesignA interface foi construída com foco na experiência do usuário e estética moderna:
Glassmorphism: Uso de transparências e desfoque de fundo (backdrop-filter) para um visual elegante.Responsividade: Layout adaptável para Desktop e Mobile utilizando Flexbox.Feedback Visual: 
Bordas coloridas que indicam o nível de prioridade (Vermelho para Alta, Amarelo para Média, Verde para Baixa).Animações: Transições suaves ao surgir novos cards na tela.🛠️ 
Tecnologias UtilizadasO sistema foi desenvolvido utilizando apenas tecnologias nativas (Vanilla), sem dependências externas, conforme as regras da disciplina:HTML5: 
Estrutura semântica (header, nav, main, section, footer).CSS3: Estilização avançada e layout responsivo.JavaScript (ES6+): Manipulação do DOM, eventos e lógica de persistência.📥 
Como Executar o ProjetoClone este repositório ou baixe os arquivos em sua máquina.Certifique-se de que os arquivos de imagem (.jpg) e o ícone animado (.gif) estejam na mesma pasta do código-fonte.
Abra o arquivo index.html em qualquer navegador moderno.
